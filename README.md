# Vortic 🌪️

**An automated AI pipeline that transforms long-form YouTube podcasts into viral, 9:16 short-form clips with burned-in subtitles, ready for TikTok, Instagram Reels, and YouTube Shorts.**

---

## How It Works

Vortic orchestrates a 4-stage pipeline that automatically handles downloading, audio extraction, semantic analysis, and precision video rendering.

```text
[ YouTube URL ]
      │
      ▼
┌─────────────┐     ┌───────────────┐     ┌───────────────┐     ┌───────────────┐
│ Downloader  │────▶│  Transcriber  │────▶│ Clip Selector │────▶│ Master Render │
│  (yt-dlp)   │     │(OpenAI Whisper│     │ (GPT-4o-mini) │     │   (FFmpeg)    │
└─────────────┘     └───────────────┘     └───────────────┘     └───────────────┘
      │                     │                     │                     │
 Download MP4       Chunk >24MB MP3       Pydantic Schema       Crop 9:16 + SRT
                    Sync Timestamps       Over-Generate x10     Burn Subtitles
```

## Tech Stack

* **FastAPI**: Lightweight, asynchronous web framework for the API layer.
* **Celery + Redis**: Background task orchestration to handle long-running video rendering jobs asynchronously without blocking the API.
* **yt-dlp**: A reliable, actively maintained YouTube downloader that easily bypasses throttling.
* **OpenAI Whisper API**: Best-in-class transcription offering precise, word-level timestamping required for subtitle generation.
* **OpenAI GPT-4o-mini**: A cost-effective LLM capable of analyzing narrative flow, finding engaging "hooks", and identifying viral moments using structured JSON.
* **FFmpeg**: The industry-standard media framework, utilized here for precision audio extraction, frame cropping, and hardcoding `.srt` subtitles natively.

## Setup & Run

### 1. Prerequisites
- Python 3.10+
- FFmpeg installed and accessible in system PATH
- Docker (for Redis)

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/arjun-1903/vortic.git
cd vortic

# Create and activate virtual environment
python -m venv venv
source venv/Scripts/activate  # On Windows: .\venv\Scripts\Activate.ps1

# Install dependencies
pip install -r requirements.txt

# Configure Environment Variables
echo "OPENAI_API_KEY=sk-your-key-here" > .env
```

### 3. Start the Pipeline
You need three terminal windows to run the stack locally.

```bash
# Terminal 1: Start Redis
docker-compose up -d

# Terminal 2: Start the Celery Worker
celery -A app.main.celery_app worker --loglevel=info --pool=solo

# Terminal 3: Start the FastAPI Backend
uvicorn app.main:app --reload
```

## Engineering Decisions & Known Limitations

* **Whisper 25MB Limit & Chunking**: The OpenAI Whisper API rejects audio files larger than 25MB. Vortic solves this by extracting a heavily compressed MP3 (`-q:a 2`), checking the file size, and dividing it into 30-minute chunks if it exceeds 24MB. The chunks are processed sequentially, and the backend recalculates precise timestamp offsets to seamlessly stitch the final transcript back together.
* **Re-encoding vs. Stream Copy**: While using `ffmpeg -c copy` is instantly fast, it restricts cuts to keyframe boundaries, causing severe subtitle desynchronization and abruptly cut words. Vortic deliberately sacrifices rendering speed by forcing full re-encoding (`libx264`) to guarantee true frame-accurate precision.
* **LLM Structured Outputs & Over-Generation**: GPT-4o-mini is constrained using Pydantic schemas to return exact start/end timestamps. Because smaller LLMs struggle with basic arithmetic (often hallucinating clip durations), Vortic uses an "Over-Generation" pattern. It asks the LLM for 10 clips, mathematically discards any that fall outside the strict 15-90s window, snaps the remaining floats to exact Whisper segment boundaries to prevent dropped words, and returns the top 5 by score. This avoids expensive API retry loops.
* **File System Sanitization**: Clip titles generated by the LLM are strictly sanitized (removing apostrophes, pipes, colons, and slashes) to prevent OS-level path errors and FFmpeg command-line injection during the render stage.
* **Center Crop Limitation**: FFmpeg currently applies a static `crop=ih*9/16:ih` to extract the vertical center of the 16:9 video. *Limitation*: If the speaker walks off-center, they will be cut out of the frame. Future iterations will require AI Face Tracking (e.g., OpenCV/MediaPipe) to dynamically pan the crop box and keep the subject in frame.

## Cost Per Video Estimate

Vortic is highly cost-optimized by utilizing targeted models for specific tasks. Processing a standard **1-hour podcast**:

* **Whisper Transcription**: ~$0.36 (60 mins * $0.006/min)
* **GPT-4o-mini Selection**: ~$0.01 (~15,000 prompt tokens, ~500 completion tokens)
* **Total**: **~$0.37 per hour of video** 
*(Subsequent runs on the same video cost $0.00, as transcripts are locally cached).*
